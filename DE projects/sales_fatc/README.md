# Построение DM модели данных

__Суть проекта:__ есть таблица на stg в postgres с наименованием orders. Я хочу разложить ее на dimensional modeling по Кимбалу в форме звезды. Пусть таблица фактов будет sales_fact, а другие будут таблицы справочники. 

Я подготовлю физическую и концептуальную модель в sqldbm, а ddl и insert операции вложу в отдельные sql файлы.
